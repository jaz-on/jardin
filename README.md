# Jardin

Un thème WordPress minimaliste pour cultiver ses idées sur le web.

## Dépôt secondaire : Jardin Events

Le plugin **[jardin-event](https://github.com/jaz-on/jardin-event)** est le dépôt associé pour la partie **événements** (CPT, blocs, motifs). Il est référencé comme dépendance de développement dans `composer.json` et complète le thème côté fonctionnalités « événements ».

- Dépôt : https://github.com/jaz-on/jardin-event  
- Installation locale des devs : `composer install` (récupère `jaz-on/jardin-event` via le dépôt VCS déclaré dans Composer).

## Installation

1. Téléchargez le thème dans `/wp-content/themes/`
2. Activez via Apparence > Thèmes
